# nickmeriano.com

-   **Framework**: [Next.js](https://nextjs.org/)
-   **Deployment**: [Vercel](https://vercel.com)
-   **Styling**: [Chakra UI](https://chakra-ui.com/)

## Running Locally

```bash
$ git clone https://github.com/nicmeriano/nickmeriano.com.git
$ cd nickmeriano.com
$ npm i
$ npm dev
```
